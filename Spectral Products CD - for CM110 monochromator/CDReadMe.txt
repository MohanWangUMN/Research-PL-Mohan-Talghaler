SP SPECTRAL PRODUCTS SOFTWARE INSTALLATION
VERSION 1.0
[March 14, 2001]

Requirements for the Software:
·	Any IBM compatible machine with an 80386 processor or higher.
·	A hard disk with at least 10MB free space, and 7MB is needed for installation.
·	A high-density 1.44MB floppy drive.
·	Microsoft Windows version 95 (Not NT) or newer in standard or enhanced mode.
·	Minimum of 32 MB of RAM recommended.

Software:
	LabView 5.1
	Visual Basic 4.0
	Quick Basic (Dos)

Installation Instructions:
	When downloading any LabView programs and you do not have LabView 
	on your system you must Download LabView RunTime Engine.

	All programs listed in the main menu will be downloaded in your Program Files directory.
	A shortcut will be created in your Start Menu/Programs directory.
